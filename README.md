Сценарии заполнения и отправки формы
Открыта форма для записи на курс “Инженер по тестированию”


Ф-1 Отправка полностью заполненной формы
1.Заполнить поле “Имя” 8-ю символами на кириллице


- Поле заполнено введенными символами на русском языке

- Нет сообщения об ошибке
2.Заполнить поле  “Телефон”  восьмизначным телефонным номером, используя цифры 0-9
- Поле заполняется введенными цифрами
- Сообщения об ошибке нет
3. Заполнить поле “Электронная почта” : Адрес состоит из двух частей, разделенных символом «@». В левой части имя почтового ящика. В правой части -доменное имя сервера, на котором расположен почтовый ящик. 
- Поле заполняется введенным адресом электронной почты

- Сообщения об ошибке нет
4. Нажать на кнопку “Записаться”
- Появляется сообщение:
об успешной записи на курс и предложение перейти к оплате












Ф-2 Отправка пустой формы
1. Оставить все поля пустыми.
2. Нажать на кнопку “Записаться”
- Форма не отправляется
- Появляется сообщение об ошибке
Ф-3 Отправка формы с незаполненным полем “Имя”
1. Поле “Имя” оставить пустым
2. Поле “Телефон” заполнить валидными данными
3. Поле “Электронная почта “ заполнить валидными данными
4.Нажать на кнопку “Записаться”
- Форма не отправляется
- Появляется сообщение об ошибке
Ф-4
1. Поле “Имя” заполнить валидным значением
2. Поле “Телефон” оставить пустым
